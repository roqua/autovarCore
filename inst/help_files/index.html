AutovarCore

AutovarCore generates networks given a data frame. AutovarCore is a simplified version of the Autovar.

To install, type the following:

install.packages('devtools')
require('devtools')
install_github('roqua/autovarCore')

For more information on Autovar, see autovar.nl.
